# Strong Password Checker

Example 1:

```
Input: password = "a"
Output: 5
```

Example 2:

```
Input: password = "aA1"
Output: 3
```

Example 3:

```
Input: password = "1337C0d3"
Output: 0
```

**Constraints:**

- 1 <= password.length <= 50
- password consists of letters, digits, dot '.' or exclamation mark '!'.
